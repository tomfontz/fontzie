## FAQ

#### How do I import it into CSS?

```css
/* MUST BE OPENTYPE!! */

/* Normal */
@font-face {
    font-family: Font-Family;
    src: url("path.to/Font") format("opentype");
}

/* Bold */
@font-face {
    font-family: Font-Family;
    font-weight: bold;
    src: url("path.to/Font") format("opentype");
}

```
